About Book Corners

Book Corners is a community map for little free libraries and neighborhood book exchange spots. The goal is simple: help people discover nearby places where they can leave a book or pick one up.

Every library submitted by users is reviewed before publication, so the map stays useful, trustworthy, and welcoming to everyone.

How to contribute

You can contribute by submitting new libraries, improving location details, and reporting outdated entries so the map keeps improving over time.

  • Add a new library in your neighborhood
  • Improve address details and marker position
  • Report places that are missing or no longer active

About the creator

Hi, I am Andrea Grandi and I've been writing software for over 30 years. I built Book Corners as a practical side project to combine community tools, geospatial data, and a love for books into something useful.

I enjoy creating practical products that start simple and improve with real feedback from people using them. This project follows that spirit: small iterations, clear priorities, and community value.

Get involved

Explore the map, submit a library in your area, or follow the code on GitHub if you want to track updates and future improvements.
